# Issues and Errors

If you experience any issues or errors, we are here to help. 

Please report technical issues via [GitHub issues](https://github.com/hyra-io/Discord-Webhook-Proxy/issues). We will try to resolve them as soon as possible. Your patience is appreciated.

If you require support please join our friendly [Discord Community](https://discord.gg/hyra) where our community will be happy to help you.
